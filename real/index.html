Fuzzy Path of Mine

I've been programming on and off since my third year of high-school. Started with Pascal and the thrill that came from seeing the computer do exactly as I tell it to, whether right or wrong. Continued with MATLAB while studying mechanical engineering. Somehow I got tangled up with Linux, CLI and vim, then distro-hopped until Ubuntu settled with me. Picked up touch-typing along the way – my current speed is around 100 WPM. Learned some C for my Master's Thesis that focused on the variants of linear and circular interpolations for real-time control of a plasma cutter via microcontroller.

During my PhD studies, some projects were overwhelming, shoved me neck deep. But, I persevered – it was too much fun to quit. It started along the lines of "You know some C? Great, now you'll learn C++ and use it together with OpenGL." Afterwards, I just kept pushing. Learned Python on a hunch, seemed cool enough. Indeed, it supercharged my abilities and my selftaught learning pace, since I was freed from the rigor of the C++ compiler. Through several Python projects, I've learned OOP, Git and a few good programming practices after getting clogged down in my own spaghetti code, tangled with a total lack of version control. Then I got back into C++ and learned it properly, with constness, less segfaults and redness from the compiler.

Currently, I'm sharpening my skills at LeetCode, getting better by the day. Also, I took up learning Java since Sedgewick's algorithms book is written in it, making it easier to follow through.
